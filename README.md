# ASP.NET Core 搭建的个人博客

本地配置步骤：
1、运行 IDentityServer 项目，需要在 Program 中修改您自己的运行地址和端口
2、运行 PostHost 项目，需要在 Program 中修改您自己的运行地址和端口，同时在配置文件中修改您存储文章的路径
3、使用 IIS 或其他代理软件代理前端
